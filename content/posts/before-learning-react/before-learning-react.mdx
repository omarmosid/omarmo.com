# 5 things to learn before diving into React in 2021

React.js is a UI library created by Facebook that lets you build your frontend UI that is declarative and composable.
It has seen tremendous growth over the last few years and has become something of a standard bearer in the world of web development.

Companies that use react have also spiked in the last few years. Among job applicants react is considered
to be a highly valuable skill and organizations are more than happy to dish out an attractive salary to folk who know react.

It is clear that learning react has a tremendous return on investment and I cannot recommend it enough, especially to those who are just starting out in the tech industry.

Having said that, i t is rather unfortunate that people rush into learning react without spending time on learning the prerequisites. I've heard from multiple friends and colleagues who've confessed to struggle through it because of a lack of knowledge on topics ranging from the confusing destructuring syntax to immutability in functional programming.

With that said, here is a list of topics I feel are essential to get a good grasp of before diving deep into react.

### 1. HTML/CSS Basics

HTML and CSS are the bread and butter of web development and you are likely to encounter them regardless of whether you are learning React or any other frontend library.

Topics to know:

1. HTML Tags
2. CSS Basics - Selectors and Tags
3. CSS Flex
4. CSS Grid

Resources

1. [W3Schools](https://www.w3schools.com/html/default.asp)
2. [FreeCodeCamp](https://www.freecodecamp.org/learn/responsive-web-design/)

### 2. Javascript Basics

Javascript has grown massively in the last few years and it's growth can be best summarized by this famous quote by Jeff Atwood

> “Any application that can be written in JavaScript, will eventually be written in JavaScript.

- Jeff Atwood, CE 2007

Javascript can be challenging if you have no background in programming. However like any other programming language it has a common set of concepts and syntaxes that can be mastered with a little bit of practice.

Topics to know:

1. Adding Javascript to HTML
2. Document Object Model
3. Variables and Expressions
4. Conditionals
5. Loops
6. Functions
7. Objects and Classes

Resources

1. [FreeCodeCamp](https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/#basic-javascript)
2. [Brad Traversey's Crash Course](https://youtu.be/hdI2bqOjy3c)
3. [Mosh Hamedani's Crash Course](https://youtu.be/W6NZfCO5SIk)

### 3. Javascript Advanced: The weird parts

Anybody who knows Javascript knows how tricky it can be to learn some of its more advanced features

Topics to know

1. Hoisting
2. Destructuring
3. Arrow functions
4. Copy by Value/Copy by Reference
5. Closures
6. Call, Apply, Bind
7. Callbacks
8. Promises
9. Async Await
10. Prototypal Inheritance
11. This Keyword
12. Try Catch and error handling
13. Event capturing and bubbling

### 5. Programming Paradigms

1. Object Oriented programming
2. Functional Programming
3. SOLID principles

Resources

1. [Object-oriented JavaScript for beginners](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/Object-oriented_JS)
2. [What is Functional Programming?](https://medium.com/javascript-scene/master-the-javascript-interview-what-is-functional-programming-7f218c68b3a0)
3. [Applying SOLID To React](https://medium.com/docler-engineering/applying-solid-to-react-ca6d1ff926a4)
