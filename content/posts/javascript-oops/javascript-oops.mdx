## Interface

An interface in typescript allows us to define the stucture/shape/syantax of a class that implements it.
This also means that a class that implements an interface is contractually bound to implment all its members.

```js
interface Person {
  firstName: string;
  lastName: string;
  getFullName(): string;
}

// Implementation
class Foo implements Person {
  firstName: string;
  lastName: string;
  getFullName(): string {
    return `My name is ${this.firstName} + ${this.lastName}`;
  }
}
```
